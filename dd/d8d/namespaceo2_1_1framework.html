A callback definition for a hook to be invoked when processes terminate

The parameter is the nullptr if the process is the main driver, for all child processes, the id string is passed. This allows to customize the hook depending on the process. Note that the callback hook is invoked for every process, i.e. main driver and all childs.

void customize(o2::framework::OnWorkflowTerminationHook& hook)

    if (idstring == nullptr) {
      std::cout << "hook" << std::endl;

    }
  };
}
