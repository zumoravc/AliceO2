Detailed Description

The analysis framework defines a number of common data types for usage in the detector modules, like e.g. kAliHLTAnyDataType and kAliHLTDataTypeDDLRaw. Those data types always have origin kAliHLTDataOriginAny. The correct detector origin can be set by using operator '|' 

AliHLTComponentDataType dt=kAliHLTDDLRawDataTypeID|kAliHLTDataOriginTPC
