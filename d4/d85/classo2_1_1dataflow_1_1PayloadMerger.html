Detailed Description

template<typename ID>
class o2::dataflow::PayloadMerger< ID >

Helper class that given a set of FairMQMessage, merges (part of) their payload into a separate memory area.

  • Append multiple messages via the aggregate method
  • Finalise buffer creation with the finalise call.

This merges a set of messages sharing the same id to a unique buffer , so that it can be either consumed or sent as a message itself. The decision on whether the merge must happen is done by the constructor specified policy mCheckIfComplete which can, for example, decide to merge when a certain number of subparts are reached. Merging at the moment requires an extra copy, but in principle this could be easily extended to support scatter - gather.
