Detailed Description

template<typename CHARGESUM>
class o2::mch::raw::BareElinkDecoder< CHARGESUM >

Main element of the MCH Bare Raw Data Format decoder.

A BareElinkDecoder manages the bit stream for one Elink.

Bits coming from parts of the GBT words are added to the Elink using the append() method and each time a SampaCluster is decoded, it is passed to the SampaChannelHandler for further processing (or none).
