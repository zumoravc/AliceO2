Detailed Description

Example task for PHOS digits monitoring.

Author
Dmitri Peresunko after Markus Fasel
Since
Dec 14, 2019

Example payload for workflows using o2::phos::Digit as payload. Printing several digit-related information for each digit. Refer to run for the list of input spec to be specified.
