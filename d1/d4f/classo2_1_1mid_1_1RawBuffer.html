◆ skipOverhead()

template<typename T >
void o2::mid::RawBuffer< T >::skipOverhead ( )

This function must be called after reading a block of data if the payload is not provided and/or readout in bytes.

In this case, indeed, there can be an overhead between the last useful bit and the declared memory size, which is in bytes. Calling this function allows to jump directly to the next header when needed
