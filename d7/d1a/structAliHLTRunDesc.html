Detailed Description

Event descriptor. The struct is sent with the SOR and EOR events.

Note
The name of the member fRunType is a bit misleading. This is not the ALICE Run Type given by the ECS to the sub-system. The member is an internal HLT run type and a combination of the HLT running mode and the beam type. 
Bit 0-2:   beam type identifier
Bit 3-31:  HLT mode
