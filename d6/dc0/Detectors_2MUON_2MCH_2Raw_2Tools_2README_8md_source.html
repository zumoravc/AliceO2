5 # MCH RawData Tools
6 
7 ## Raw Dump
8 
9 The `o2-mchraw-dump` executable reads events from a raw data file and computes
10 the mean and sigma of the data of each channel.
11 
12 The data format (Bare or UserLogic) is deduced from the data itself.
13 
14 Until the data is completely self-consistent though (when we'll move to RDH v5
15 or 6), you'll have to provide the CRU id with the `--cru` option.
16 
17 ```bash
18 > o2-mchraw-dump -h
19 Generic options:
20  -h [ --help ] produce help message
21  -i [ --input-file ] arg input file name
22  -n [ --nrdhs ] arg number of RDHs to go through
23  -s [ --showRDHs ] show RDHs
24  -j [ --json ] output means and rms in json format
25  --cru arg force cruId
26 
27 > o2-mchraw-dump -i $HOME/cernbox/o2muon/data-DE617.raw -j -n 10000 -cru 0
28 ```
29 
30 Note that options `-j` and `-s` are incompatible (as the rdh output is not in
31 json).
