Detailed Description

template<typename InterfacePolicy = DefaultInterface, typename InitializerPolicy = default_initializer, typename PrinterPolicy = default_printer>
class gNeric::RuntimeContainer< InterfacePolicy, InitializerPolicy, PrinterPolicy >

the technical base of the mixin class

for the mixin class The class is necessary to provide the innermost functionality of the mixin.

The level of the mixin is encoded in the type 'level' which is incremented in each mixin stage.
