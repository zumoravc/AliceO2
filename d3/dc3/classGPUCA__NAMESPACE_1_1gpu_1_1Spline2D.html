Detailed Description

template<typename DataT, int nFdimT = 0, bool isConsistentT = 1>
class GPUCA_NAMESPACE::gpu::Spline2D< DataT, nFdimT, isConsistentT >

The main Spline class. Contains constructors and interpolation.

F dimensions can be set as a template parameter (faster; the only option for the GPU) or as a constructor parameter (slower; the only option for the ROOT interpretator). In a compiled CPU code one can use both options.
